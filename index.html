Styling CSS Tools

Code (example) Example Value Name Description Values
color: #00FFFF; Cyan or Aqua Colour. Hexademical Colour It is a number system that has sixteen digits, 0 to 9 followed by “A” to “F”. Hex values always begin with # and specify values of red, blue, and green using hexadecimal numbers such as #00FFFF (aqua colour). Six-digit hex values with duplicate values for each RGB value can be shorted to three digits.
color: rgb(0,255,255); Cyan or Aqua Colour. RGB RGB colors use the rgb() syntax with one value for red, one value for blue and one value for green. RGB values range from 0 to 255.
color: hsl(180, 100%, 50%) Cyan or Aqua Colour. HSL HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is). Hue ranges from 0 to 360 and saturation and lightness are both represented as percentages.
opacity: 0.5; 50% visible Opacity
font-family: 'Times New Roman'; Times New Roman Font Family Describes the font of an element. A big variety!
font-weight: 700; 700 (bold) Font Weight It controls how bold or thin text appears. It can be specified with keywords or numerical values. Numerical values can range from 1 (lightest) to 1000 (boldest), but it is common practice to use increments of 100. A font weight of 400 is equal to the keyword value normal, and a font weight of 700 is equal to bold.
font-style: italic; italic Font Style How the text appears.
text-transform: uppercase; uppercase Text Transformation If the letters will be shown as uppercase or lowercase. uppercase or lowercase.
letter-spacing: 2px; 2 pixels Letter Spacing It is used to set the horizontal spacing between the individual characters in an element.
word-spacing: 0.3em; 0.3em Word Spacing It sets the space between words.
line-height: 1.4; 1.4 Line height It set how tall we want each line containing our text to be. Can be a unitless number, such as 1.4 , or a length value such as 12px, 5% or 2em. The unitless value is preferred since it is responsive based on the current.
text-align: right; right Text Alignment It aligns the text to its parent element. Right or left.
